# Contribute on SkateHive

## Run locally 

```bash
git clone https://github.com/SkateHive/nextskateapp.git

cd nextskateapp 

pnpm i 

pnpm dev 

```


## In addition, you will need:

HIVE Wallet: 
[💼 How to create a HIVE Wallet - Skatehive Docs](https://docs.skatehive.app/docs/Level%20-%202/hive-wallet)
